Parsing Pipeline

解析链路与支持格式

平台将不同来源文件统一接入同一条解析链，形成摘要、结构化信息、证据块和可索引文本。

支持的主流格式

PDF TXT / MD DOCX CSV / JSON HTML / XML XLSX / XLS PNG / JPG / JPEG WEBP / GIF / BMP
  • 文本类文件优先走原生 UTF-8 与结构化解析。
  • Office 与表格类文件抽取正文、表头、字段和关键信号。
  • 图片与扫描类文件支持 OCR 或图片元数据保留。

统一解析步骤

01

快速解析

识别格式、抽取正文、生成摘要与初始分类建议。

02

结构化

提取字段、标签、实体、证据块和文档画像。

03

详细解析

对高价值文档继续做深解析与补充识别。

04

索引就绪

进入文档库、向量索引、长期记忆和后续输出链。
