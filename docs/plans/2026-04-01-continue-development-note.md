## 明确可对外介绍的技术原理

### 文档解析

- 支持格式：
  - `pdf`
  - `txt` / `md`
  - `docx`
  - `csv` / `json`
  - `html` / `htm` / `xml`
  - `xlsx` / `xls`
  - `png` / `jpg` / `jpeg` / `webp` / `gif` / `bmp`
- 解析链路：
  - 快速解析
  - 自动摘要
  - 自动分组建议
  - 深度结构化
  - 索引与记忆同步

### 向量化与检索

- 记录层包括：
  - `summary`
  - `profile`
  - `profile-field`
  - `evidence`
  - `claim`
- 检索策略是多阶段：
  - 规则命中
  - 向量召回
  - 重排筛选
- 普通问答优先使用目录与记忆，精确事实或正式输出时补更多文档详情
