# AI 数据分析中台

基于 OpenClaw 定制的本地化、只读型企业 AI 数据分析中台。

## 目标

- 只读接入企业数据源（文档 / 数据库 / 网页后台 / API）
- 通过网页工作台进行 IM 式问答
- 输出企业经营分析、知识问答、结构化摘要、可视化报表
- 尽量限制系统能力为只读，避免对客户系统产生写入影响

## 当前内容

- `docs/PRD.md`：产品需求文档
- `docs/ARCHITECTURE.md`：技术架构与模块拆分
- `apps/web`：前端工作台骨架
- `apps/api`：后端 API 骨架
- `apps/worker`：扫描 / 索引 / 报表任务骨架
- `packages/*`：数据源、报表、Agent 核心模块骨架

## 推荐开发顺序

1. 先完成文档接入（PDF/合同/技术文档）
2. 再完成数据库只读查询
3. 再补网页后台只读抓取
4. 最后补权限、审计、报表中心

## 运行方式（当前为骨架）

```bash
pnpm install
pnpm dev
```

> 目前仓库内主要是结构与文档，便于继续迭代，不是完整可运行产品。
