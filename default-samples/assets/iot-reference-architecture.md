# Internet of Things System Reference Architecture

This sample document summarizes a vendor-neutral IoT system reference architecture and is used as a default knowledge base example.

Core layers:
- device connectivity
- edge gateway processing
- cloud ingestion and storage
- analytics and application services

Representative capabilities:
- sensor data collection
- event routing and alerting
- device lifecycle management
- semantic interoperability
- API integration for enterprise systems

Recommended usage:
- use for architecture planning
- use for deployment mode comparison
- use for integration interface mapping
