# 智慧仓储 IOT 解决方案样例

项目目标：为区域仓配中心建设一套可复制的智慧仓储 IOT 解决方案，实现设备接入、库存感知、异常预警、能耗监测和可视化运营。

系统架构：
- 设备层：温湿度传感器、门磁、烟雾报警器、电子货架标签、AGV小车、视频网关。
- 边缘层：工业网关负责 Modbus、MQTT、RS485 协议接入与本地缓存。
- 平台层：统一设备管理、规则引擎、告警中心、数据中台、API 服务。
- 应用层：仓储看板、移动巡检、库存预警、设备运维工单。

部署模式：边缘计算网关 + 云端平台双层部署。  
接口类型：MQTT 设备上报、RESTful 查询接口、Webhook 告警推送。  
集成方式：对接 ERP、WMS、企业微信告警、数据看板系统。

关键指标：
- 库存盘点准确率 >= 99.5%
- 设备在线率 >= 98%
- 异常告警响应时间 <= 5 分钟
- 人工巡检成本下降 30%

适用场景：多仓协同、冷链仓储、电子产品仓储、自动化拣货场景。
